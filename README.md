# 物料仓库管理

个人物料/元器件管理系统，支持云端同步。

## 功能特性

- 📦 物料管理：添加、编辑、删除物料
- 🔍 快速搜索：通过名称、型号、分类搜索
- 📊 库存管理：入库、出库数量记录
- ☁️ 云端同步：支持 Supabase 云存储和云端备份/恢复
- 💻 桌面应用：基于 Tauri 的原生桌面体验

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 配置 Supabase 云存储（可选）

1. 访问 [Supabase](https://supabase.com) 创建免费账户
2. 创建新项目，获取 `URL` 和 `anon key`
3. 在 Supabase SQL 编辑器中执行：

```sql
-- 创建物料表
create table materials (
  id bigint generated by default as identity primary key,
  name text not null,
  model text,
  category text not null,
  description text,
  quantity integer default 0,
  location text,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null,
  updated_at timestamp with time zone default timezone('utc'::text, now()) not null
);

-- 启用实时订阅
alter publication supabase_realtime add table materials;
```

4. 如果需要网页端云端备份和多电脑登录恢复，再创建用户备份表：

```sql
create table if not exists warehouse_backups (
  id uuid primary key references auth.users(id) on delete cascade,
  user_id uuid not null references auth.users(id) on delete cascade,
  snapshot jsonb not null,
  updated_at timestamptz not null
);

alter table warehouse_backups enable row level security;

-- 启用分类、位置、排序等整份备份的实时订阅
alter publication supabase_realtime add table warehouse_backups;

create policy "Users can read own backup"
on warehouse_backups for select
using (auth.uid() = user_id);

create policy "Users can create own backup"
on warehouse_backups for insert
with check (auth.uid() = user_id and auth.uid() = id);

create policy "Users can update own backup"
on warehouse_backups for update
using (auth.uid() = user_id)
with check (auth.uid() = user_id and auth.uid() = id);
```

5. 在 Supabase Authentication 中启用 Email 登录方式。

6. 配置 Supabase 连接，二选一：

- 发布前创建 `.env`，写入 `VITE_SUPABASE_URL` 和 `VITE_SUPABASE_ANON_KEY`
- 或打开应用后，在“云端设置”中填写 Supabase `URL` 和 `anon key`

```env
VITE_SUPABASE_URL=https://你的项目.supabase.co
VITE_SUPABASE_ANON_KEY=你的 anon public key
```

### 3. 运行开发模式

```bash
npm run tauri dev
```

### 4. 构建生产版本

```bash
npm run tauri build
```

## 技术栈

- **前端**: React + TypeScript + Vite
- **桌面**: Tauri
- **云存储**: Supabase

## 使用说明

1. **添加物料**：点击"添加物料"按钮，填写物料信息
2. **搜索物料**：在顶部搜索框输入关键词
3. **分类筛选**：点击左侧分类列表筛选
4. **库存管理**：点击 +1/-1 按钮调整数量
5. **云端账号**：点击“登录云端”注册或登录邮箱账号
6. **云端备份**：点击“上传云端”保存当前账号的完整快照，换电脑后登录同一账号并点击“从云端恢复”

## 注意事项

- 首次使用未配置 Supabase 时，数据保存在浏览器本地存储
- 配置 Supabase 后，数据会自动同步到云端
- 支持离线使用，云端连接后会同步数据
